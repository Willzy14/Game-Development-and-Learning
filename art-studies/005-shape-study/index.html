Art Study #5: Shape Study

Applying 5-value lighting theory to different 3D primitives

Lighting Principles Applied to Each Shape

Cube

• Flat planes = sharp value transitions

• Each face has uniform value

• Hard edges between planes

Cylinder

• Curved surface = gradient like sphere

• Flat ends = uniform value

• Linear highlight along length

Cone

• Gradient narrows toward apex

• Terminator curves around form

• Sharp tip = value convergence

Torus

• Double curvature = complex gradient

• Inner shadow from self-occlusion

• Highlight wraps around tube
