### 2. Composition Theory (CRITICAL)

#### Core Concepts to Learn:
- **Rule of Thirds**: WHY it works (eye movement, balance)
- **Golden Ratio**: Phi (1.618), spiral composition
- **Leading Lines**: Directing viewer's eye through the scene
- **Visual Flow**: Path the eye takes through composition
- **Negative Space**: Empty space as compositional element
- **Focal Point Math**: 80/20 rule - where to concentrate effort
- **Visual Weight**: Balance without symmetry

### 3. Color Theory (Deep Dive)

### 7. Game Art Techniques (INDUSTRY SPECIFIC)

#### Core Concepts to Learn:
- **Focal Point in Games**: Directing eye during gameplay
- **Parallax Layers**: Depth through scrolling speeds
- **Readability**: Art that serves mechanics (clear hitboxes)
- **Gameplay Clarity**: Never sacrifice function for beauty
- **Visual Hierarchy**: Player > interactive > background
- **Color Coding**: Using color for game information
- **Silhouette Readability**: Recognizable at any angle

- **Hollow Knight**: Environmental storytelling, atmospheric depth
- **Hades**: Character readability in chaos
