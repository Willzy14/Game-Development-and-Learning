## Game Description
Classic Pong game with two paddles and a ball. Player controls left paddle with W/S keys, AI controls right paddle. First to 5 points wins.

## Core Mechanics
1. Paddle movement (up/down)
2. Ball physics (velocity, bouncing)
3. Collision detection (ball vs paddles, ball vs walls)
4. Score tracking
5. Basic AI opponent
6. Win condition
